An Amazing plugin for manage your socials feeds with your own wordpress site.

== Description ==

Keep Your Site Looking Fresh: Automatically push your new Instagram content straight to your site to keep it looking fresh and keeping your audience engaged.
Save Time: Don't have time to update your photos on your site? Save time and increase efficiency by only fetch posts per #hashtag, Username and displaying them on your website.
There are some Tempplates and Layouts for that, such as: Masonry, Grid, Wall and Carousel.

Check demo version: <a href="https://www.ithemelandco.com/Plugins/Social-Feed-Assistant/">Pro Version</a>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After Install and Activate this plugin a menu will be added in your admin menu named “Feed Assistant” that is included 2 submenus as:
a-	Dashboard: Display the plugin details.
b-	Social Feed Assistant: Add your Feeds, Streams, Layouts and then used the generated shortcode.
